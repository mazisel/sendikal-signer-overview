# Sendikal Signer

**🇹🇷 Türkçe**

EYP ve CAdES e-imza işlemleri için geliştirilmiş Windows tabanlı yerel imza ajanı.

## Tech Stack
- .NET 8 (WPF + Kestrel)
- Windows Certificate Store
- PKCS#11 (AKIS)

## Özellikler
- Yerel HTTP servisi ile web uygulamalarına imza desteği
- CAdES detached imza üretimi
- Kullanıcı onaylı güvenli imza süreci
- AKIS token entegrasyonu

---

**🇬🇧 English**

A Windows-based local signing agent developed for EYP and CAdES electronic signature processes.

## Tech Stack
- .NET 8 (WPF + Kestrel)
- Windows Certificate Store
- PKCS#11 (AKIS)

## Features
- Local HTTP service for signature support to web applications
- CAdES detached signature generation
- User-approved secure signing process
- AKIS token integration
